	<h2>简介</h2>
	<p>创立于2009年的太原理工大学清泽心雨主题网站，始终致力于构建网络文化育人长效机制和工作环境，现已成为沟通学校与学生的桥梁，引领大学生生涯发展的航标，展现百年学府风采的窗口，汇聚莘莘学子的网上精神家园。《人民日报》（2014年08月14日）称赞“清泽心雨”已成为全国知名的网络文化育人引擎。</p>
<p>网站首页采用最前沿的响应式网站布局技术，秉承简约美学的设计原则，通过渐近增强与优雅降级的方式，对统一DOM元素完美实现了多终端与多浏览器的视觉呈现。无论是导航响应渐显渐动、要闻随鼠标hover放大，还是名片式微电影动效，都为用户缔造出优雅悦目的交互动效体验。
着力构筑注重育人实效的网络平台，牢牢掌握网络教育阵地的主导权、控制权，在网络育人实践层面发挥示范性作用，在思政网站建设理论层面取得创新性成果。</p>
<p>站群整合资讯、论坛、博客、微博、微视、学工服务、电子商务等系统，组成新闻中心、综合媒体集团、清泽微视集团等集团，建立起理念创新、功能全面、资源丰沛的网络生涯导航体系。现有十个子站，85个一级栏目，319个二级栏目，416个三级栏目，60余个特色专题。每年上传新闻稿件、博文评论、学习资料、照片视频十万篇以上。
网站总点击累计近千万人次，目前日均点击11000人次，最高日点击近十万次，百度收录52万3000篇，谷歌收录34万5000篇，搜狗26万9000篇。来访网友遍及120余国家和地区。
把网络文化育人作为网络思想政治教育工作攻坚克难的关键抓手，发挥网络文化活动抵近前沿、引领思想、滋润心灵、陶冶情操的独特效能。</p>
<p>清泽心雨开展科技竞赛基地巡礼活动，展示课外科技成果，努力打造百科全书式课外科技平台；清泽小讲堂“求索论坛”鼓励学生走上讲台，探索和打造网络文化建设的特色品牌；在全校范围内多次主办主题摄影活动，记录“时间都去哪了”的瞬间；携手KAB俱乐部，开展了以“情系校友、探索职场”校友访谈活动，引导大学生规划生涯。
始终坚持以用户体验为中心，以学生需求为导向，精心开办实学、实用、实益的特色栏目和应用，增强学生对主题网站使用的黏度和交联度，不断提高网络文化产品创新能力及服务水平。
在“口述历史”节目里，网站记者走访老干部、老教师，对这些见证历史的耄耋老人谈话录影，抢救性搜集校史资料。
在“梦想在路上”专题里，通过访谈毕业十年左右、事业小成的创业校友，与读者分享其成败得失、苦辣酸甜。
在火爆的“校长奖学金”网络评选里，短短十天间候选人事迹材料总点击近五十万次，让学习标兵真正成为学习偶像。</p>
<p>清泽心雨网站高度重视网络产品的创新创意能力，以赛促研，积极组队参加多项全国高校赛事，在“挑战杯”、公益广告征集活动、“中软杯”软件大赛、大广赛中屡获奖项，成绩斐然。
聚焦创新驱动的顶层设计，发挥市场在网络资源配置中所起的“决定性”作用，以更宽广的视野谋划和带动网站技术持续进步。
不懂市场规则的思政网站，在学生心目中一定没有“市场”，必定被边缘化为互联网海洋中的信息孤岛。清泽心雨以商业网站为师，更加重视用户个性化服务与体验，利用市场竞争倒逼机制科学规划网站建设。
清泽心雨成立新媒体研究院以致力于新媒体产业研究、新媒体内容研究、新媒体技术研究。下设技术委员会和内容建设委员会，设立网络文化研究、舆情调研、网络安全、移动互联网应用、智慧校园等十个大学生网络文化研究室。现承担一项教育厅思政处主持的严肃游戏开发项目，一项省高等学校哲学社会科学研究课题研究。
坚持学校主导与校内外多元参与相结合，培养造就信得过、靠得住、打得赢的网络文化育人生力军。
团队建设是清泽心雨赖以发展传续的坚强基石。</p>
<p>四年来，先后有208名教师、辅导员进论坛、进博客，逾千名同学直接参与网站的建设，“团结奉献、民主包容、追求卓越、开拓创新”的文化认同和精神认同，影响并砥砺着每一位清泽站友展示文化自信、实现文化自强。
围绕建设网络生态文明下功夫，营造健康向上的校园网络舆论氛围，构筑风清气正的绿色网络空间。
构建第一课堂、第二课堂和网络课堂三位一体, 线上与线下双向互动，虚拟世界与现实世界交汇融合的网络生态系统。
净化网络环境，抑制网络负能量，强化网络舆情收集，培育广大师生开展经验交流、理论探讨、感悟分享的“文明、绿色、和谐”的网络社区，有效发挥正面宣传鼓舞人、激励人的作用。
当前清泽心雨正加快推进网站迭代创新与重构，创建数字全媒体网络文化育人系统。</p>
<p>清泽心雨整合支持多终端发布、多内容采集、多模式应用等一体化应用的数字“全媒体”，建设运用即时通讯、移动互联网、云计算等新兴技术的思政云导航系统，促进网络教育资源利用层次不断提高。
今后清泽心雨将大力弘扬社会主义核心价值观，引导广大青年学生凝心聚力实现最美中国梦！</p>
